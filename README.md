# 🏗️ CreaTech Precast AI

An AI-powered decision support system for **precast concrete manufacturing optimization**, designed to predict cycle time, estimate production cost, assess de-mould risk, and recommend optimal curing strategies using machine learning.

---

## 🚀 Overview

CreaTech Precast AI leverages **XGBoost models**, feature engineering, and explainable AI techniques to optimize precast production processes. The system evaluates multiple manufacturing scenarios and recommends the most efficient combination of curing methods, automation levels, and admixtures.

---

## ✨ Features

- ⏱️ **Cycle Time Prediction**
  - Predicts curing and production cycle duration.

- 💰 **Cost Prediction**
  - Estimates total manufacturing cost per cubic meter.

- ⚠️ **Risk Classification**
  - Detects premature de-moulding risks.

- 🧠 **Explainable AI**
  - SHAP-based feature importance and model interpretation.

- 🔄 **Optimization Engine**
  - Evaluates numerous configurations and recommends the best strategy.

- 📊 **Interactive KPI Dashboard**
  - Plotly-based visualizations for production insights.

- 📈 **Exploratory Data Analysis**
  - Multi-panel statistical analysis of the dataset.

- 📁 Model Export
  - Saves trained models and generated reports.

---

## 🏗 Problem Statement

Precast concrete manufacturing is affected by:

- Climate conditions
- Material composition
- Curing methods
- Automation levels
- Project requirements

Traditional approaches rely heavily on experience and manual decision-making, leading to:

- Longer cycle times
- Increased production costs
- Quality inconsistencies
- Higher risk of premature de-moulding

CreaTech Precast AI aims to solve these challenges through data-driven optimization.
